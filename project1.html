Make your Web Page

[1] The Basics of Web and HTML

1. Web pages are composed with HTML. Understanding the basic syntax of HTML is needed to make a web page.

2. Using tags, we can insert images/links and change plain text into bold or italic.


3. The set of opening tag, content and closing tag is called an "element". There are inline and block elements. The block elements makes invisible boxes while inline don't.

4. We can add style by using CSS. And the whole HTML document has a structure, which we would be learning in future lectures.

[2] Creating a Structured Document with HTML

1. Elements of web page
HTML - structure
CSS - style
JAVA - interactive component

2. Web page are made up by boxes
We can add description of boxes by naming the class and add the style in the CSS file (box width, height, margin, location etc)
- Basic structures: head, body (span, div containers)

[3] Adding CSS Style to HTML Structure

1. Link the style.css file in the head element
link rel="stylesheet" href="style.css"
*css file have to be in the same folder with html

2. Using Selector properly, we can avoid repetition
example: ID, class, tag, attribute, positional
- When changing the style, we can change the specific components simultaneously.
- Inheritance: text properties are automatically applied in the inferior components unless the style is overrided

3. Box model
We can define the content, padding, border and margin. By using "box-sizing:border-box;", the size includes the border and padding (not the margin).
Flexbox enables to arrange boxes (block element) side by side.

Summary

1. Think of the structure! Draw all the boxes.
2. Think about the repeated style and semantic elements.
3. Write your HTML code.
4. Add the style starting from bigger to smaller components.
