Project 1: Make your Web Page

[1] The Basics of Web and HTML

I. Web pages are composed with HTML. Understanding the basic syntax of HTML is needed to make a web page.

II. Using tags, we can insert images/links and change plain text into bold or italic.

III. The set of opening tag, content and closing tag is called an "element". There are inline and block elements. The block elements makes invisible boxes while inline don't.

IV. We can add style by using CSS. And the whole HTML document has a structure, which we would be learning in future lectures.

[2] Creating a Structured Document with HTML

I. Elements of web page

  1. HTML - structure
  2. CSS - style
  3. JAVA - interactive component

II. Structure of HTML

  • DOCTYPE html on the first line declared that the document is a HTML document
  • Some tags are used to describe the structure: html, head, title, body, h1 (heading), p (paragraph)
  • We indent while writing codes to make the structure of HTML easily visible: tree structure - able to expand or collapse, encapsuled

III. Web page are made up by boxes

  • Add description of boxes by namming the class

[3] Adding CSS Style to HTML Structure

I. Link the style.css file in the head element

<link rel="stylesheet" href="style.css">

*css file have to be in the same folder with html

II. Using Selector properly, we can avoid repetition

  • example: ID, class, tag, attribute, positional

III. Box model

  • We can define the content, padding, border and margin.
  • By using "box-sizing:border-box;", the size includes the border and padding (not the margin).
  • Flexbox enables to arrange boxes side by side.
